# Flow-simulation-parallel

---
### Сборка проекта
```shell
mkdir cmake-build
cd cmake-build
cmake ..
cmake --build .
```
### Запуск 

- Исходная версия алгоритма ```./original```
- Версия с оптимизациями алгоритма ```./monothread```
- Многопоточная версия оптимизированного алгоритма ```./multithread```
